# bevy_asset_preload
Provides a plugin for the bevy game engine to preload all assets in the assets folder and automatically switches to the next state when done.

## Versions
What version of this library supports which bevy version?

| bevy_asset_preload version | bevy version |
|----------------------------|--------------|
| 1.0.0                      | 0.16         |
| 1.1.0                      | 0.17         |
